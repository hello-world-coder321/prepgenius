# 🚀 PrepGenius

An AI-powered study and preparation platform designed to help students prepare smarter, track progress, and improve consistently.

---

## 📌 Overview

PrepGenius is a smart learning system that helps students preparing for competitive exams like **JEE, NEET, UPSC**, etc. It focuses on:

- 📚 Structured syllabus tracking  
- 📅 Daily study planning  
- 🔁 Spaced repetition & revision scheduling  
- 📊 Performance tracking & weak area detection
